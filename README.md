# Stocks
Hello! My name is James, and this is my randition of a stock bot!
Only partly relying on statistics, I wanted to heavily rely on news and social media to make a decision on whether or not to buy certain stocks.
With the beta version out, I have trained a model to detect patterns across stock prices of 50 different companies using aspects such as returns, volatility, and moving averages.
I have also created a scraper for popular online forums (I originally was going to use Instagram, but credibility was an issue) such as reddit and stocktwits, while also looking at authortative sources such as EDGAR. Unfortunately, Reddit requires manual approval for access to their APIs, so I am still waiting on that.
I then made a scraper for newsAPI with the caveat that all the articles are a day behind.
I put all these sections on a website where a user can search up any one of 50 companies (to be expanded), and will get summaries from the sources listed above.
